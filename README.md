# Laphzaeraash – Interactive Dictionary Prototype

## Overview
**Laphzaeraash** is an interactive prototype dictionary web app for an my native language, which is unstandardised. It demonstrates how to build a **searchable, multimedia dictionary interface** using **HTML, CSS, JavaScript, and JSON**.  

This project was **developed with the assistance of ChatGPT (AI)**, which provided guidance on coding, debugging, design, and structuring the project. All final implementation, decisions, and styling were performed manually to create a professional, functional prototype.

---

## Features
- **Dynamic Word Cards**
  - Words are loaded from a JSON file (`Words/Sheen.json`)
  - Each card displays:
    - Word & part of speech
    - Meaning
    - Example sentence
    - Optional notes
    - Optional audio pronunciation
- **Search Functionality**
  - Real-time search filters cards dynamically
- **Responsive Design**
  - Works on mobile and desktop
- **Dark Green Themed Styling**
  - Modern typography and hover effects
  - Logo displayed in the header
- **Multimedia Support**
  - Audio pronunciation per word

---

## Technologies Used
- HTML5 – structure and layout  
- CSS3 – styling, responsive design, hover effects  
- JavaScript – fetching JSON, dynamic DOM manipulation, search functionality  
- JSON – storing dictionary words and metadata  
- Audio (.mp3) – pronunciation files  
- Local Server (Live Server / Python HTTP Server) – required for fetch()
